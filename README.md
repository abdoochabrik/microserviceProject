# microserviceProject
This project contains 7 interconnected softwares:

1. API Gateway (Spring Cloud): [GitHub link](https://github.com/abdoochabrik/freelenceEasyApiGateway)
2. Discovery Server (Spring Cloud, Netflix Eureka): [GitHub link](https://github.com/abdoochabrik/freelanceEasyDiscoveryServer)
3. User Microservice (Spring Boot): [GitHub link](https://github.com/abdoochabrik/userMicroservice)
4. Parent Scrapper Microservice (NestJS): [GitHub link](https://github.com/abdoochabrik/freelanceEasyParantScrapper)
5. Child Scrapper Microservice (Spring Boot): [GitHub link](https://github.com/abdoochabrik/freelanceEasyChildScrapperMicroService)
6. Shell scripts to set up Docker containers, Kafka cluster, and Keycloak server: [GitHub link](https://github.com/abdoochabrik/freelanceProjectScriptFiles)
7. Frontend (Angular): [GitHub link](https://github.com/abdoochabrik/freelanceEasyFRonEnd)
